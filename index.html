2- Qual a diferença ente git e github?
R= Enquanto o Git é um software, o GitHub é uma plataforma onde você pode subir seus projetos, criar seu portfólio e compartilhá-lo com o mundo.
O GitHub funciona usando o Git para que os programadores possam desenvolver e atualizar os seus projetos sozinhos ou de forma colaborativa com outros programadores.

3- o que faz o comando commit?
R= Os commits são as unidades estruturais de um cronograma de projeto Git.
Podem ser considerados instantâneos ou marcos ao longo do cronograma de um projeto Git

4- O que faz o comando push?
R= O comando push transfere commits do repositório local a um repositório remoto.
